เกรดเฉลี่ย: 3.18

บริษัท ทีมพรีซิชั่น จำกัด (มหาชน)

ตำแหน่งงาน: Desing & Test Development

รายละเอียดงาน: ออกแบบเครื่อง Tester, ออกแบบวงจร, เขียนโปรแกรมควมคุมเครื่อง Tester, เขียนโปรแกรมอื่นๆตามที่ได้รับมอบหมาย, ปรับปรุงและพัฒนา Hardware & Software, Support Production Line

รายละเอียดงานเชิงลึก

ความสามารถด้าน Programmer

-สามารถเขียนโปรแกรม Windows Application ได้ โดยใช้ Visual Studio ภาษา C# ระดับเชี่ยวชาญ เคยเขียนและใช้งานจริงมาแล้ว มีผลงานต่างๆ มากมาย

-สามารถแก้ไขโปรแกรมที่เขียนด้วยภาษาต่างๆ ได้ เช่น Python, VB, C++, Arduino เป็นต้น

-สามารถเขียนโปรแกรมเชื่อมต่อและใช้งานฐานข้อมูล MySQL ได้ แต่ยังไม่เคยใช้งานจริง

-ปัจจุบันกำลังฝึกฝนการเขียนโปรแกรมฝั่ง FrontEnd สามารถใช้ HTML CSS JavaScript ได้
